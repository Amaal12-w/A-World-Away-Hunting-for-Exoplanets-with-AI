Hunting for Exoplanets

WHAT WE DO

We develop an AI-powered educational platform that analyzes real NASA datasets (Kepler, K2, TESS) to detect and classify exoplanets. Users can upload observational data or manually enter parameters (orbital period, transit duration, planetary radius, etc.) to get an AI classification: Confirmed Planet, Candidate, or False Positive. Visualizations, model accuracy stats, and easy educational explanations help both researchers and learners explore findings.

Features & Data

🔭 Real NASA Data

Support for Kepler, K2 and TESS datasets (open-source CSVs). Users can upload files for batch analysis.

🤖 AI Analysis

Machine learning pipelines preprocess transit variables, train models, and output classification probabilities.

📊 Visualizations

Interactive charts, transit plots, and model accuracy dashboards to explain results visually.

⚙ User Controls

Manual data entry, optional hyperparameter tuning, and downloadable result reports for researchers.

🎓 Educational Content

Lesson plans, videos, and explanation widgets that simplify exoplanet science for students.

🌍 Habitability Focus

Special modules that highlight habitability indicators and prioritize promising candidates.
